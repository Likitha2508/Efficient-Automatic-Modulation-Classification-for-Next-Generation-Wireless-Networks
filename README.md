# Efficient Automatic Modulation Classification for Next-Generation Wireless Networks

This repository contains the implementation of the paper:

**Efficient Automatic Modulation Classification for Next-Generation Wireless Networks**

The project focuses on Automatic Modulation Classification (AMC) using deep learning techniques for wireless communication systems.

---

## Datasets

The implementation uses the following publicly available RadioML datasets from DeepSig:

### 1. RadioML 2016.10A
- Dataset Link:  
  https://www.kaggle.com/datasets/nolasthitnotomorrow/radioml2016-deepsigcom
- Description:  
  A synthetic dataset generated using GNU Radio consisting of 11 modulation types at different Signal-to-Noise Ratio (SNR) levels. :contentReference[oaicite:0]{index=0}

### 2. RadioML 2018.01A
- Dataset Link:  
  https://www.kaggle.com/datasets/pinxau1000/radioml2018
- Description:  
  A large-scale dataset containing 24 digital and analog modulation types with both simulated channel effects and over-the-air recordings. :contentReference[oaicite:1]{index=1}
